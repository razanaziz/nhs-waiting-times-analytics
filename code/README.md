# Dataset Information

This dataset combines NHS operational and demographic variables used throughout the dissertation analysis.

## Dataset Characteristics
- 72 variables
- NHS England operational data
- Time period: 2016–2023

## Included Variables

### Operational Metrics
- A&E waiting times
- Total attendances
- 4-hour performance targets
- Bed occupancy levels
- Workforce statistics

### Demographic Variables
- Age groups
- Gender distributions
- Ethnicity categories
- Deprivation indices

### Hospital Capacity Variables
- Total available beds
- Occupied beds
- Emergency workforce size

## Analytical Applications

This dataset was used for:
- regression analysis
- correlation analysis
- SARIMA forecasting
- operational healthcare analytics
- demographic performance assessment
